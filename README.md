#  Expense Tracker project

##  Description

This is a simple and efficient command-line based Expense Tracker built using Python to help users manage their daily expenses. The application allows users to add expenses with categories and view all recorded transactions along with the total spending.

##  Features

* Add new expenses with amount and category
* View all recorded expenses in a structured format
* Automatic calculation of total expenses
* Input validation and error handling for better user experience
* File-based storage for persistent data

##  Technologies Used

* Python
* File Handling

## 🎯 Purpose

This project demonstrates practical implementation of Python fundamentals, including file operations, exception handling, and user interaction, to solve a real-world financial tracking problem.

##  How to Run

1. Run the Python script
2. Choose options from the menu
3. Add or view expenses easily
